**Antaria** — Sistema de tandas (savings circles) on-chain con interfaz en WhatsApp. Digitaliza el modelo tradicional mexicano de ahorro colectivo usando smart contracts para transparencia y automatización.

## 🚨 Errores y Blockers

### 1. Complejidad del modelo de tandas
- **Problema:** Las tandas tienen reglas complejas (turnos, montos, garantías) que deben codificarse en un smart contract
- **Desafío:** Manejar defaults y dispute resolution on-chain

### 2. Integración WhatsApp
- **Problema:** Conectar un bot de WhatsApp con smart contracts en Monad
- **Nota:** Esta integración requiere un backend intermediario

## 📈 Curva de Aprendizaje
- **Nivel inicial:** Intermedio — entiende el modelo de tandas pero nuevo en smart contracts
- **Principal desafío:** Traducir lógica financiera cultural (tandas) a código inmutable
- **Progreso:** Diseño conceptual avanzado
- **Lección clave:** Las tandas son un caso de uso perfecto para smart contracts pero la UX vía WhatsApp es clave para adopción en LATAM
